# Preview Project Workflow

This workflow details the commands required to run the local development server or preview a production build of the Next.js application.

## Steps

1. **Start Development Server**
   Start the Next.js development server:
   ```bash
   npm run dev
   ```
   *Note: This server runs at `http://localhost:9002` by default as configured in `package.json`.*

2. **Start Production Preview**
   If you want to run the production build locally:
   ```bash
   npm run start
   ```
