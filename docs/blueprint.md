# **App Name**: CleanPro Connect

## Core Features:

- Service Provider Homepage: A public-facing home page providing information about the National Cleaning Company and the opportunity for independent service providers to join, including brand details.
- Provider Registration Portal: A secure process for new independent service providers to register and apply to work with the company.
- Personalized Provider Dashboard: A dedicated account/profile area for each service provider to view and manage their assigned work.
- Schedule and Job Management: Displays the service provider's upcoming schedule, including details of assigned customer cleaning jobs.
- Accounting and Payments View: Provides service providers with a clear overview of their earnings, invoices, and payment statuses.
- Internal Messaging System: Enables direct communication between service providers, clients, and company administrators.
- AI-powered Messaging Assistant: An AI-powered tool that suggests quick replies or generates draft responses for common inquiries within the messaging system, improving communication efficiency.

## Style Guidelines:

- The chosen palette aims for a fresh, professional, and reliable feel, suitable for a cleaning service. The primary color is a deep, professional blue (#1F70B2) which evokes trust and cleanliness. The background features a very light, desaturated blue-grey (#ECF2F7) to maintain an airy and clean aesthetic without being stark white. For contrast and important calls to action, an analogous and vibrant aqua accent (#19CCCC) is used.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean lines and excellent readability across various text lengths and screen sizes, ensuring clarity and professionalism for all service provider interactions.
- Utilize a consistent set of minimalist, line-based icons throughout the application. These icons should be clear, intuitive, and support quick recognition of features such as scheduling, messaging, and financial reporting, reinforcing a modern and efficient user experience.
- Employ a clean, structured, and organized layout with ample whitespace to enhance readability and reduce cognitive load. Dashboards should be modular, allowing for clear separation of information such as upcoming jobs, messages, and earnings, optimized for efficient navigation and task management.
- Implement subtle, functional animations for state changes and user interactions. Examples include smooth transitions between sections, gentle loading indicators, and small, non-distracting feedback animations for successful actions like message sending or schedule updates, improving perceived responsiveness.
